- block : 화면을 수직 분할(행을 나눔) + width / height 속성 사용 가능
        - inline : 화면을 수평 분할 + width / height 속성 사용 불가능
        - inline-block : inline의 수평분할 + block 크기 조정
        - none : 화면에 요소가 표시되지는 않으나 존재하고 있는 상태
        - flex : 요소의 정렬되는 방향, 요소간의 간격을 유연하게 처리하는 형식
